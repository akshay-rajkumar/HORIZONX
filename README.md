# CineVault / HorizonX

Luxury movie streaming frontend with Express backend (TMDB proxy). Single server serves both API and static site.

## Running locally

```bash
cp .env.example .env
# Edit .env and set TMDB_API_KEY (get one at https://www.themoviedb.org/settings/api)
npm install
npm start
```

Open http://localhost:3001 — the app and player are served from the same origin.

## Public hosting & security

- **Static files** live in `public/` only. The server serves **only** this folder; `.env`, `server.js`, and `node_modules` are never exposed.
- **Secrets**: Use a `.env` file (copy from `.env.example`). Never commit `.env` (it’s in `.gitignore`).
- **Security**: Helmet sets safe HTTP headers; `express-rate-limit` limits API requests (60/min in production). API route params (movie/tv/genre ids) are validated as positive integers.
- **Production**: Set `NODE_ENV=production` and configure your host (e.g. Railway, Render, VPS) to run `npm start` and set `PORT` and `TMDB_API_KEY` in the environment.
